Oracle 替换字符串

发布于 2021-04-28 | 分类于 Oracle 、 问题整理 | 3分钟 | 641字数
函数说明
regexp_replace(source_char, pattern, replace_string,position,occurrencr,match_parameter)

# source_char 搜索的值

# pattern 正则表达式

# replace_string 替换字符串,反向引用子表达式\n

# position 开始检索的字符的位置，默认为1

# occurrence 替换的位置，0 替换所有的匹配项，如果为n(正整数)，替换第n个匹配项

# match_parameter 匹配的行为 
# i 指定不区分大小写的匹配
# c 区分大小写的匹配
# n 允许句点'.'与换行符，默认将源字符串视为单行
# x 忽略空格字符，默认情况，空白字符与自身匹配
# 如果指定多个矛盾值，将按照最后一个值，如ic，区分大小写

#match_parameter 不指定，将默认为下面的操作：
# 1. 区分大小写 2. 句点与换行符不匹配 3. 单行
示例
select regexp_replace('0123456789', '01234', 'oabc') from dual; # 0abc56789

select regexp_replace('0123456789', '[0-9]', '*#') from dual; # *#*#*#*#*#*#*#*#*#*#

select regexp_replace('01234567abcfsg89', '[a-z]') from dual; # 0123456789

select regexp_replace('+86 15153090946','(\+[0-9]{2})( )([0-9]{3})([0-9]{4})([0-9]{4})', '(\1)\3-\4-\5') from dual; # (+86)151-5309-0946

select regexp_replace('012as56','(.)','\1 ') from dual; #0 1 2 a s 5 6

select regexp_replace('0asca98', '[0-9a-zA-Z]', '*', 4) from dual; # 0as****

select regexp_replace('0asc98', '[0-9a-zA-Z]', '*', 4, 3) from dual; # 0asc9*

select regexp_replace('0asC98','[A-Z]', '*', 1, 0, 'i') from dual; #0***98

select regexp_replace('a
b
c
d
e
f', '([a-z])(.)','\1', 1, 0, 'n') from dual; # abcdef

select regexp_replace('aa a a bb b b','a a', '0') from dual;  # a0 a bb b b

select regexp_replace('aa a a bb b b', 'a a', '0', 1, 0, 'x') from dual; # 0 a a BB b b
正则表达式
符号 释义
\
* 匹配0个或多个
+ 匹配1个或多个
? 匹配0个或1个
| 或运算，所有均可视为单独的子表达式
^ 默认情况下匹配字符串的结尾，多行模式匹配字符串每一行的行尾
$ 默认情况下匹配字符串的开头，多行模式匹配每一行的行头
. 字符集支持的任意字符，NULL除外
[] 指定匹配列表
() 对表达式分组，视为单个子表达式
{m} 匹配m次
{m,} 匹配至少m次
{m,n} 匹配至少m次，但不超过n次
\n 反向引用表达式
[::] 指定字符类
字符类
[:XXXX:] 含义 范围
[:space:] 包括换行、空格、tab在内的空白字符 tab 空格
[:blank:] 空格与tab tab 空格
[:alpha:] 字母 a-z A-Z
[:alnum:] 字母与数字 a-z A-Z 0-9
[:digit:] 十进制数字 0-9
[:xdigit:] 十六进制数 0-9 a-f A-F
[:lower:] 小写字母 a-z
[:upper:] 大写字母 A-Z
[:cntrl:] 控制字符 ctrl、backspace等
[:punct:] 标点符号 ,.?' 不确定中英文、半角全角
